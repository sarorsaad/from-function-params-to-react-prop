شرح الأقواس المعقوفة { } في JSX 🎯

الأقواس المعقوفة { } هي الطريقة التي نخبر بها React: "هذا كود JavaScript، قم بتنفيذه"

1. الاستخدام الأساسي للأقواس المعقوفة 📌

// 1. مع المتغيرات البسيطة
const name = "أحمد";
<div>مرحباً {name}</div>

// 2. مع العمليات الحسابية
const age = 25;
<div>عمرك بعد 5 سنوات: {age + 5}</div>

// 3. مع الشروط (ternary)
<div>
    {age >= 18 ? "بالغ" : "قاصر"}
</div>

2. استخدام الأقواس المعقوفة مع الخصائص (Props) 🔄

// يمكن استخدام { } في خصائص العناصر
const buttonText = "انقر هنا";
const isDisabled = true;

<button
    disabled={isDisabled}
    className={isDisabled ? 'disabled-btn' : 'active-btn'}
>
    {buttonText}
</button>

3. استخدام الأقواس المعقوفة مع map 🗺️

const items = [
    { id: 1, name: "عنصر 1" },
    { id: 2, name: "عنصر 2" }
];

<div>
    {items.map(item => (
        <div key={item.id}>
            {item.name}
        </div>
    ))}
</div>

متى نستخدم { }؟ 🤔

  • عند استخدام متغيرات JavaScript
  • عند إجراء عمليات حسابية
  • عند كتابة شروط (ternary operations)
  • عند استخدام دوال مثل map
  • في خصائص (props) العناصر

أخطاء شائعة ⚠️

❌ خطأ 
<div className={active-class}>
// يجب أن تكون "active-class" داخل علامات اقتباس
✅ صحيح 
<div className={"active-class"}>

نصائح مهمة 💡

  • لا تنسَ إغلاق الأقواس المعقوفة
  • تأكد من استخدام علامات الاقتباس للنصوص داخل { }
  • يمكن استخدام أي تعبير JavaScript صالح داخل { }
  • لا يمكن استخدام عبارات if/else داخل { }، استخدم العمليات الشرطية بدلاً منها
