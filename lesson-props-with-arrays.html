شرح Props مع المصفوفات في React

في هذا الدرس، سنتعلم كيفية استخدام Props مع المصفوفات في React لعرض قائمة من البيانات بشكل ديناميكي.

١ تعريف مكون التحية (Greeting Component)

نقوم بإنشاء مكون يستقبل مصفوفة من الأشخاص كـ prop ويعرض معلوماتهم: 

function Greeting({ people }) {
    return (
        <div>
            <h3>قائمة الأشخاص:</h3>
            {people.map((person, index) => (
                <div key={index}>
                    مرحباً {person.name}، عمرك {person.age} سنة.
                </div>
            ))}
        </div>
    );
}
شرح الكود:
  • نستخدم { people } للحصول على المصفوفة من الـ props
  • نستخدم map للتكرار على كل عنصر في المصفوفة
  • نستخدم key لتعريف كل عنصر بشكل فريد (مطلوب في React)

٢ إنشاء مصفوفة البيانات

نقوم بإنشاء مصفوفة تحتوي على كائنات تمثل بيانات الأشخاص: 

function App() {
    const peopleList = [
        { name: "أحمد", age: 25 },
        { name: "سارة", age: 30 },
        { name: "محمد", age: 35 },
        { name: "نور", age: 28 }
    ];

    return <Greeting people={peopleList} />;
}
نقاط مهمة:
  • كل شخص يمثل بكائن يحتوي على خاصيتي name و age
  • نمرر المصفوفة كاملة كـ prop باسم people

٣ النتيجة النهائية

عند تشغيل التطبيق، سنرى النتيجة التالية:

قائمة الأشخاص:

مرحباً أحمد، عمرك 25 سنة.
مرحباً سارة، عمرك 30 سنة.
مرحباً محمد، عمرك 35 سنة.
مرحباً نور، عمرك 28 سنة.

٤ المميزات الرئيسية

  • إعادة الاستخدام: يمكن استخدام نفس المكون مع بيانات مختلفة
  • الديناميكية: يمكن تغيير البيانات بسهولة دون تغيير المكون
  • الصيانة: سهولة تحديث وصيانة الكود
  • القابلية للتوسع: يمكن إضافة المزيد من الأشخاص بسهولة

نصائح مهمة:

  1. دائماً استخدم key فريد عند استخدام map في React
  2. حاول تنظيم البيانات في مصفوفة من الكائنات لسهولة التعامل معها
  3. استخدم أسماء واضحة للـ props والمتغيرات

٦ تمارين عملية

جرب التعديلات التالية على الكود:

  2. أضف خاصية جديدة لكل شخص (مثل المدينة أو الوظيفة)
