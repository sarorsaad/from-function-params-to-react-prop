شرح Map: من JavaScript إلى React 🚀

سنتعلم كيف نستخدم دالة map في JavaScript وكيف نطبقها في React لإنشاء واجهات ديناميكية.

1. نقطة البداية: دالة map في JavaScript العادي 📚

// لدينا مصفوفة أرقام
const numbers = [1, 2, 3, 4];

// استخدام map بشكل عادي في JavaScript
const doubledNumbers = numbers.map(number => number * 2);
// النتيجة: [2, 4, 6, 8]

2. الدالة السهمية (Arrow Function) ⚡

// الطريقة التقليدية
numbers.map(function(number) {
    return number * 2;
});

// الطريقة الحديثة (arrow function)
numbers.map(number => number * 2);

3. المعاملات (Parameters) في map 📝

// معامل واحد (العنصر)
array.map(item => ...)

// معاملان (العنصر والفهرس)
array.map((item, index) => ...)

// ثلاثة معاملات (العنصر والفهرس والمصفوفة كاملة)
array.map((item, index, array) => ...)

4. استخدام map في React لتحويل المصفوفة إلى عناصر JSX 🔄

// 1. لدينا مصفوفة من البيانات
const students = [
    { id: 1, name: "أحمد" },
    { id: 2, name: "سارة" },
    { id: 3, name: "محمد" }
];

// 2. نستخدم map لتحويلها إلى عناصر JSX
function StudentsList() {
    return (
        <div>
            {students.map((student, index) => (
                <div key={student.id}>
                    {index + 1}. {student.name}
                </div>
            ))}
        </div>
    );
}

5. مثال تدريجي: من JavaScript إلى React 🎯

// خطوة 1: مصفوفة عادية
const fruits = ["تفاح", "موز", "برتقال"];

// خطوة 2: استخدام map في JavaScript
const fruitsList = fruits.map(fruit => "الفاكهة: " + fruit);

// خطوة 3: استخدام map في React
function FruitsList() {
    return (
        <div>
            {fruits.map((fruit, index) => (
                <div key={index}>
                    {index + 1}. الفاكهة: {fruit}
                </div>
            ))}
        </div>
    );
}

6. لماذا نضع map داخل { } في JSX؟ 🤔

في React/JSX، نستخدم { } للتعبير عن الكود JavaScript داخل HTML. هي طريقة لنقول لـ React: "هذا كود JavaScript، قم بتنفيذه"

مثال تدريجي 🔄

// 1. هذا HTML عادي
<div>Hello</div>

// 2. هذا JSX مع متغير JavaScript
const name = "أحمد";
<div>{name}</div>

// 3. هذا JSX مع map (لأنها كود JavaScript)
<div>
    {courses.map(course => (
        <div>{course.name}</div>
    ))}
</div>

تشبيه بسيط 🌟

// الرسالة بدون متغيرات
<div>
    مرحباً، كيف حالك؟
</div>

// الرسالة مع متغير
const name = "أحمد";
<div>
    مرحباً {name}، كيف حالك؟
</div>

// الرسالة مع قائمة أسماء (map)
const names = ["أحمد", "محمد", "سارة"];
<div>
    مرحباً:
    {names.map(name => (
        <div>{name}</div>
    ))}
</div>

لماذا نحتاج { }؟ ⚠️

// ❌ خطأ
<div>
    courses.map(course => ...)
</div>

// ❌ خطأ
<div>
    {courses.map(course => ...)}
    {/* نسيان key */}
</div>

// ✅ صحيح
<div>
    {courses.map(course => (
        <div key={course.id}>
            {course.name}
        </div>
    ))}
</div>

مثال شامل 📝

function CourseList() {
    const courses = [
        { id: 1, name: "React", students: 20 },
        { id: 2, name: "JavaScript", students: 15 }
    ];

    return (
        <div>
            {/* عنوان ديناميكي */}
            <h1>{"قائمة المساقات (" + courses.length + ")"}</h1>

            {/* قائمة المساقات باستخدام map */}
            {courses.map(course => (
                <div key={course.id}>
                    {/* اسم المساق */}
                    <h2>{course.name}</h2>
                    
                    {/* عدد الطلاب مع شرط */}
                    <p>{course.students > 15 ? "صف ممتلئ" : "متاح التسجيل"}</p>
                </div>
            ))}
        </div>
    );
}

تذكر دائماً 🎯

نستخدم { } في JSX لـ:

  • تنفيذ كود JavaScript
  • عرض قيم المتغيرات
  • تنفيذ العمليات الحسابية
  • كتابة الشروط
  • استخدام دوال مثل map

أخطاء شائعة وتصحيحها ⚠️

// ❌ خطأ
<div>
    courses.map(course => ...)
</div>

// ❌ خطأ
<div>
    {courses.map(course => ...)}
    {/* نسيان key */}
</div>

// ✅ صحيح
<div>
    {courses.map(course => (
        <div key={course.id}>
            {course.name}
        </div>
    ))}
</div>

⭐ النقاط المهمة للتذكر

  1. دالة map تعمل فقط على المصفوفات
  2. كل عنصر في map في React يحتاج إلى key
  3. يمكنك استخدام index كمعامل ثاني إذا احتجت إلى ترقيم
  4. النتيجة النهائية هي مصفوفة من عناصر JSX
