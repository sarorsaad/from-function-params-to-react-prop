شرح Map في React - تكرار العناصر بسهولة 🔄

تعتبر دالة map من أهم الأدوات في React للتعامل مع المصفوفات وعرض العناصر المتكررة بشكل ديناميكي.

🌟 المثال الأساسي

// لدينا مصفوفة من الأسماء
const names = ["أحمد", "محمد", "سارة"];

function NamesList() {
    return (
        <div>
            {names.map((name, index) => (
                <div key={index}>{name}</div>
            ))}
        </div>
    );
}

🤔 لماذا نحتاج map؟

بدون map، سنضطر لكتابة: 
return (
    <div>
        <div>{names[0]}</div>
        <div>{names[1]}</div>
        <div>{names[2]}</div>
    </div>
);

هذه الطريقة:

  • ✖️ متعبة
  • ✖️ غير مرنة
  • ✖️ لا تعمل مع المصفوفات ذات الطول المتغير

📝 مثال عملي مع بيانات أكثر تعقيداً

const students = [
    { id: 1, name: "أحمد", grade: 85 },
    { id: 2, name: "سارة", grade: 90 },
    { id: 3, name: "محمد", grade: 95 }
];

function StudentsList() {
    return (
        <div className="students-list">
            {students.map((student) => (
                <div key={student.id} className="student-card">
                    <h3>الاسم: {student.name}</h3>
                    <p>الدرجة: {student.grade}</p>
                </div>
            ))}
        </div>
    );
}

🎨 مثال مع تنسيقات مختلفة

⭐️ النقاط المهمة للتذكر

  1. دائماً استخدم key 
    const items = [
    { id: '1', name: 'Item 1' },
    { id: '2', name: 'Item 2' }
];

{items.map((item) => (
    <div key={item.id}>
        {item.name}
    </div>
))}
  2. يمكنك استخدام الشروط داخل map 
    {items.map((item) => (
    item.visible && 
    <div key={item.id}>
        {item.name}
    </div>
))}
  3. يمكنك استخدام map مع التنسيقات 
    {items.map((item) => (
    <div 
        key={item.id}
        className={item.status === 'active' ? 'active' : 'inactive'}
    >
        {item.name}
    </div>
))}

🚫 أخطاء شائعة وحلولها

1. نسيان الـ key

❌ خطأ 
{items.map((item) => (
    <div>
        {item.name}
    </div>
))}
✅ صحيح 
{items.map((item) => (
    <div key={item.id}>
        {item.name}
    </div>
))}

2. استخدام map خارج الـ return

❌ خطأ 
const elements = items.map(...)
return <div>{elements}</div>;
✅ صحيح 
return (
    <div>
        {items.map(...)}
    </div>
);
